In reference and respect to the amazing community at **[freeCodeCamp](https://www.freecodecamp.org)**, I have embarked on the adventure to make history with the **[Python Certification](https://www.freecodecamp.org/learn/python-v9/)** track.

## Aim of Project
This project demonstrates practical steps taken to achieve mastery in Python data types, by building a simple report card printer.

### How To Run Program
- Dowonload the repo files in your local machine
- Open your terminal; with the shorthand command `ctrl+alt+t`
- Access the directory with the command `cd report_card_printer`
- Edit the Python file with Vim (VS Code or PyCharm): `vi main.py`
- You could play with the file or leave in the default state
- Run the file by inputting the command: `python3 main.py`
